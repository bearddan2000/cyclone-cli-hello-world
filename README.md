# cyclone-cli-hello-world

## Description
A POC for cyclone programming language.

## Tech stack
- g++
- make

## Docker stack
- ubuntu:18.04
- rzuckerm/cyclone

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
